# 零杯-珞珈任意门

## 项目主题

> 武大作为武汉旅游必来之地，有许多优美的景色。春天的樱花大道洁白如雪，恍若仙境；傍晚的樱顶晚霞绚丽，美轮美奂；雄伟的老图书馆庄重典雅，底蕴丰厚；引人入胜的万林博物馆弘扬文化，展现艺术魅力。

本项目从“**自然风光**”、“**典雅建筑**”、“**学生活动**”三个角度切入，以**门户网站**的形式展现珞珈山下的美景。

 ## 从源代码构建

 ### 安装包管理工具 npm
 ```
 npm install
 ```

 ### 启动本地服务器
 ```
 npm run dev
 ```

 ### 创建一个生产环境的构建版本
 ```
 npm run build
 ```
